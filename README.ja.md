# qrclock
The Japanese README is here: [README.ja.md](README.ja.md)

シンプルなQRクロックで、現在の日付と時刻をQRコード形式で表示します。

## デモ
[ライブデモ](https://code4fukui.github.io/qrclock/)

## 機能
- QRコード形式で現在の日付と時間を表示します
- QRコードを1秒ごとに自動的に更新します
- QRコードをクリックすることで全画面モードが利用可能です

## 使用方法
1. リポジトリをクローンしてください: `git clone https://github.com/code4fukui/qrclock.git`
2. `index.html` ファイルをウェブブラウザで開いてください。
